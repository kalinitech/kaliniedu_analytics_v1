
# KaliniEdu Analytics

A comprehensive education analytics platform for teachers and schools to process, analyze, and visualize student performance data. Built with Next.js, Supabase, Prisma, and Paystack.

## Features

- **Excel Data Upload**: Process student marksheets in Excel format
- **Automated Grading**: Multiple grading systems (CBC, KCSE, KCPE, custom)
- **Analytics Dashboard**: Visualize class performance, grade distributions, and trends
- **Subscription Management**: Flexible pricing plans with Paystack integration
- **Admin Panel**: Manage plans, grading systems, subscriptions, and analytics
- **PDF & Excel Reports**: Generate detailed performance reports

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Database**: Supabase (PostgreSQL)
- **ORM**: Prisma (for type generation)
- **UI**: shadcn/ui, Tailwind CSS, Radix UI
- **Payments**: Paystack (M-Pesa, card payments)
- **Charts**: Recharts
- **File Processing**: xlsx, jspdf, jspdf-autotable

### Prerequisites

- Node.js 18+
- npm or bun
- Supabase account
- Paystack account (for payments)


## Admin Panel

### Features:
- **Analytics**: View total uploads, revenue, active subscriptions, and monthly trends
- **Plans**: Manage subscription plans (create, edit, deactivate)
- **Grading Systems**: Define custom grading thresholds
- **Subscriptions**: Track active/pending subscriptions
- **Seed Data**: Initialize default plans and grading systems
